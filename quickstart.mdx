## 4. Rotate the Default Passwords

Protect the server by replacing default credentials immediately after the first login.

### Linux VPS

Run the `passwd` utility:

```bash
passwd
```

Provide the current password followed by a strong new passphrase (at least 12 characters with mixed case, numbers, and symbols).

### Windows VPS

1. Press **Ctrl + Alt + End** (or use the on-screen keyboard on macOS).
2. Choose **Change a password**.
3. Submit the old password, then enter and confirm the new one.
4. Press **Enter**.

Store new credentials in a secure password manager.

## 6. Configure Hostname and Reverse DNS (Optional)

Assign a descriptive hostname:
