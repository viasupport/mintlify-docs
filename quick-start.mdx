## 3. Connect to Your VPS

Once your VPS is active, connect directly using **SSH** (Linux/macOS) or **RDP** (Windows).

### For Linux VPS
Use your terminal:

```bash
ssh root@<your-vps-ip>
```

Accept the fingerprint and enter your root password from the welcome email.

### For Windows VPS
Open **Remote Desktop Connection** (mstsc) and connect to your VPS IP:

```
Computer: <your-vps-ip>
User: Administrator
```

Enter your password when prompted to access your Windows desktop.


## 4. Change Your Password

For security reasons, you should **change your default password** as soon as you log in for the first time.

### For Linux VPS

Run the following command in your SSH session:

```bash
passwd
```

Then enter:

1. Your **current password**
2. A **new strong password**
3. Confirm the new password

## 5. Update Your System

Always begin by installing the latest updates and patches.

**Debian / Ubuntu**

```bash
apt update && apt upgrade -y
```

**AlmaLinux / Rocky / CentOS**

```bash
dnf update -y
```

Keeping your system updated ensures security and stability.

## 6. Set Hostname and Reverse DNS (Optional)

Set a custom hostname for your VPS:

```bash
hostnamectl set-hostname myserver.example.com
```

To configure reverse DNS (PTR record), use the **rDNS* section in your dashboard.
A matching hostname and PTR record help improve email deliverability and server reputation.
