## Connect from Windows Desktop

<Steps>
  <Step title="Open Remote Desktop Connection">
    Press the Windows key, type <strong>Remote Desktop Connection</strong>, and launch the desktop app. Prefer a shortcut? Press <code>Win + R</code>, enter <code>mstsc</code>, then click <strong>OK</strong>.

    ![Open RDP client on Windows desktop](/assets/1sSq3fI0sZJj8Zc4RoSr4-E6s05O4r7d3ThHTglgxfu-20241202-110408.png)
  </Step>
  <Step title="Enter the VPS address">
    In the <strong>Computer</strong> field, paste the IPv4 address (or hostname) from your welcome email. Click <strong>Show Options</strong> to expand the dialogue before you connect.

    ![Enter the VPS IPv4 in the Computer field](/assets/1sSq3fI0sZJj8Zc4RoSr4-eE8Z7tBNfWJaa74qfI7Qv-20241202-112004.png)
  </Step>
  <Step title="Use the Administrator account">
    Unless you created a custom user, type <strong>Administrator</strong>. Leave ‚ÄúAsk for credentials every time‚Äù enabled if multiple people share the workstation.

    ![Enter Administrator as the username](/assets/1sSq3fI0sZJj8Zc4RoSr4-K1RXNa_wHwVjBcGOvTw4K-20241202-112943.png)
  </Step>
  <Step title="Connect and authenticate">
    Click <strong>Connect</strong>, enter the VPS password, and accept the certificate warning. Tick ‚ÄúDon‚Äôt ask me again‚Äù once you trust the fingerprint so the prompt stays hidden next time.

    ![Password prompt and certificate warning](/assets/1sSq3fI0sZJj8Zc4RoSr4-AGGOxauevnKCyRuSL3vFW-20241202-113527.png)
  </Step>
</Steps>

Once authenticated, the Windows desktop loads and you can begin managing your VPS. Close the session any time via **Start ‚Üí Sign out** or by closing the RDP window.

## macOS Client (Microsoft Windows App)

<Steps>
  <Step title="Install Microsoft Windows App">
    Download the free client (formerly Microsoft Remote Desktop) from the <a href="https://apps.apple.com/app/microsoft-remote-desktop/id1295203466" target="_blank" rel="noreferrer">Mac App Store</a>.

    ![Microsoft Windows App listing in the Mac App Store](/assets/huaNpFAxK5PVCl_NLo3ID-QlQORqnbwx4sd8x4KCHOl-20241202-160326.png)
  </Step>
  <Step title="Add your VPS">
    Open the app, click the <strong>+</strong> button, choose <strong>Add PC</strong>, and enter the VPS IPv4 address. Give it a friendly name so you recognise it later.

    ![Add PC window showing IP entry and friendly name](/assets/huaNpFAxK5PVCl_NLo3ID-rbCU-h7FzGiJeIcW7xO1y-20241202-160642.png)
  </Step>
  <Step title="Store credentials or prompt each time">
    Select <strong>Administrator</strong> (or your custom user). Save the connection and double-click the tile to launch it.

    ![Saved connection tile ready to launch](/assets/bYb9Hny4tfRDK5FcumtQA_screenshot-2024-12-02-at-171013.png)
  </Step>
  <Step title="Approve prompts and sign in">
    macOS will ask about certificates and optionally keychain storage‚Äîpress <strong>Continue</strong>. Enter the password and you‚Äôll land on the Windows desktop.

    ![Certificate prompt during first RDP connection on macOS](/assets/vI9GQIvvP61Ayho-Uhrnj_screenshot-2024-12-02-at-171118.png)
  </Step>
</Steps>

Pro tip: If you‚Äôre using a MacBook, enable ‚ÄúFit session to window‚Äù in the connection settings for a smoother Retina experience.

![Windows desktop reached via Microsoft Windows App on macOS](/assets/huaNpFAxK5PVCl_NLo3ID-W0EeJBw1Um2ua4QWM6--f-20241202-161319.png)

## Keep RDP Secure

- Rotate the Administrator password regularly and store it in a password manager.  
- Use the portal VNC console to disable/enable RDP if you ever need to lock access quickly.  
- Maintain Windows Updates (`Settings ‚Üí Windows Update`) so security patches install promptly.
