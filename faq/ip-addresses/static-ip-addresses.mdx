## Static by Default

Each VPS receives:  \n
- One dedicated IPv4  \n
- A /64 IPv6 subnet

These assignments do not rotate or change automatically.  \n+They persist through reboots, reinstalls, and plan upgrades until you cancel the service or request an IP change.
