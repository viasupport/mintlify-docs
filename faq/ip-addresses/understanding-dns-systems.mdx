## Reverse DNS (PTR)

EDBB hosts PTR records on an in-house anycast platform.  \n+Nameserver IPs are announced from multiple PoPs so reverse lookups stay fast and resilient worldwide.

## Forward DNS (Resolvers)

Default resolvers on EDBB images point to Cloudflare (1.1.1.1 / 1.0.0.1).  \n+Anycast routing sends queries to the nearest available node for low latency and built-in DDoS protection.
