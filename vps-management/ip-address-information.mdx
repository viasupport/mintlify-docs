# Review VPS Networking Details

Need the exact IP assignments or gateways for your VPS? The EDBB portal keeps everything in one panel so you always know how the server is configured.

---

## Check the Network Info Card

<Steps>
  <Step title="Log in to the portal">
    Visit <a href="https://my.edbb.com/clientarea.php" target="_blank">my.edbb.com</a> and open <strong>Services</strong>.
  </Step>
  <Step title="Select your VPS">
    Click the target service to bring up the management dashboard.
  </Step>
  <Step title="Open the Network Info section">
    Scroll to <strong>IP Settings</strong>. Expand the card to see every assigned address and gateway.
  </Step>
</Steps>

![IP Address Settings](/assets/vps-network-info.webp)

## What Youâ€™ll See

- **Primary IPv4** plus any additional addresses that belong to the VPS  
- **Dedicated /64 IPv6** block, including the auto-configured address currently active on the NIC  
- **Default gateways** for IPv4 and IPv6 so you can configure static routes inside the OS  
- **Reverse DNS status** with quick links to edit PTR records  
- **Nameserver entries** that match the configuration supplied by the hypervisor

Use this view as the source of truth whenever you reconfigure networking inside the guest operating system.
